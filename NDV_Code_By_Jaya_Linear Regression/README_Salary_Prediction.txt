
# Linear Regression: Salary Prediction

## 📌 Objective
To predict employee salary based on years of experience using simple linear regression.

## 📁 Dataset
- File: Salary_Data.csv
- Columns: `YearsExperience`, `Salary`

## 🚀 Steps
1. Load dataset using pandas
2. Visualize data using seaborn and matplotlib
3. Apply simple linear regression with scikit-learn
4. Evaluate model performance (MSE, R²)
5. Plot regression line
6. Bonus: Error bars, user input for predictions

## 🧠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- scikit-learn

## 💡 Sample Output
- R² Score ≈ 0.95+
- Predictions match actual salaries closely

## 📌 How to Run
1. Place `Salary_Data.csv` in the notebook directory.
2. Run the Jupyter notebook `Salary_Prediction_Assignment.ipynb` step by step.
3. Optionally input experience to get a salary prediction.
